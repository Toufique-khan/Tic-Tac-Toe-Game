# 🎮 Tic Tac Toe Game

A simple yet interactive **2-Player Tic Tac Toe Game** built using **HTML, CSS, and JavaScript**.

## 🚀 Features
- ✅ 2-Player Gameplay
- ✅ Winner Detection Logic
- ✅ Draw Match Detection
- ✅ New Game Functionality
- ✅ Reset Game Button
- ✅ Responsive Design using Media Queries
- ✅ Clean and Interactive UI

## 🧠 What I Learned
While building this project, I understood the core logic behind the famous **Tic Tac Toe DSA pattern**.

This project helped me learn:
- Game state management
- Conditional logic handling
- Winning pattern algorithms
- DOM manipulation
- Event handling in JavaScript
- Responsive frontend design

## 🛠️ Tech Stack
- HTML
- CSS
- JavaScript

## 📌 Project Purpose
This project was created to strengthen my understanding of:
- JavaScript fundamentals
- Real-world logic building
- Problem-solving using DSA concepts
